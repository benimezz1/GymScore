# GymScore (Beta)
Treino personalizado sem IA (baseado em score e regras), feito com HTML/CSS/JS e publicado no GitHub Pages.

## Features
- PT + EN (troca de idioma)
- Form + Perguntas finais
- Motor de score (dias, nível, foco, local, lesões, cardio)
- Tempo do cardio separado do tempo do treino
- Treino da semana + plano 5 semanas
- PDF via Imprimir/Salvar

## Publicar no GitHub Pages
1. Settings → Pages
2. Deploy from branch → `main` / root
3. Salvar → abrir o link gerado
